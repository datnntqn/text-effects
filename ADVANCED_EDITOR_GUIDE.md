### Available Effects

#### 4. **Glow**

#### 6. **Rainbow**

## âš™ï¸ Advanced Styling

### Font Size
- **Range**: 12pt - 72pt
- **Slider Control**: Precise adjustment
- **Live Preview**: See size changes
- **Recommended**: 24-36pt for social media

### Font Weight
- **Options**: Light, Regular, Medium, Semibold, Bold, Heavy, Black
- **Segmented Control**: Easy selection
- **Effect**: Changes text thickness
- **Tip**: Bold for impact, Light for elegance

### Letter Spacing (Kerning)
- **Range**: -2 to +10
- **Effect**: Space between characters
- **Negative**: Tighter spacing
- **Positive**: Looser spacing
- **Use**: Adjust for different fonts

### Line Height
- **Range**: 0.5x - 3.0x
- **Effect**: Space between lines
- **Default**: 1.2x
- **Multi-line**: Adjust for readability

### Shadow
- **Range**: 0 - 30pt radius
- **Effect**: Drop shadow depth
- **Color**: Matches text color
- **Use**: Add depth and dimension

### Text Alignment
- **Options**: Left, Center, Right
- **Effect**: Text positioning
- **Multi-line**: Affects all lines
- **Default**: Center

## ğŸ’¡ Pro Tips

### Design Tips

1. **Contrast is Key**: High contrast = better readability
2. **Less is More**: Short text works best
3. **Test Effects**: Try different effects with same text
4. **Save Often**: Save templates you like
5. **Preview First**: Always check before exporting

### Performance Tips

1. **Simpler Effects**: Faster rendering
2. **Lower Quality**: Faster exports for testing
3. **Shorter Text**: Better animation performance
4. **Close Other Apps**: Free up memory

### Creative Tips

1. **Layer Effects**: Combine glow with other effects
2. **Seasonal Themes**: Match colors to holidays
3. **Brand Consistency**: Save brand templates
4. **A/B Testing**: Try multiple versions
5. **Inspiration**: Browse template gallery
